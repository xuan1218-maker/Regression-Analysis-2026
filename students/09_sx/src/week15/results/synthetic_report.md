# Task A：合成数据与模型比较


## 数据生成过程（DGP）

**样本量和特征数：**
- 样本量：500
- 特征数：4
- 正类比例：0.408

**真实系数：**
- feature_1：β = 2.0（提高正类概率）
- feature_2：β = -1.5（降低正类概率）
- feature_3：β = 0.0（无影响）
- feature_4：β = 0.0（无影响）
- 截距项：-0.5

**数据生成机制：**
**η = Xβ + intercept**
**p = 1/(1 + e^(-η))**
**y ~ Bernoulli(p)**


## Task A：线性回归与逻辑回归对比

### 线性回归结果 (sklearn LinearRegression - 错误示范)
- RMSE：0.3669
- R²：0.4420
- 预测范围：[-0.710, 1.273]
- 超出[0,1]范围的比例：0.087
- 系数：[0.21947319269317683, -0.1991623846487479, 0.02648795107882043, 0.03120560460937504]

**线性回归的问题（为什么不能直接拿来做二分类）：**
1. 输出不在[0,1]范围内，无法解释为概率
2. 使用MSE作为损失函数，不适合分类问题
3. 将分类问题当作回归问题处理，输出没有概率意义
4. 对"错得很自信"的预测惩罚不够

### 逻辑回归结果
- 准确率：0.8467
- 预测概率范围：[0.000, 0.995]

**逻辑回归的优势：**
1. 输出严格在(0,1)范围内，可解释为概率
2. 使用对数损失（来自伯努利似然），适合分类
3. 专门为二分类问题设计

### 核心问题回答
关键区别不是"能不能分类"（两者都可以用阈值分类），而是**"输出是否有概率意义"**。线性回归输出任意实数，而逻辑回归输出的是合理的概率值。
