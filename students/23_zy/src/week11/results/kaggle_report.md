# Week 11 Task B：Kaggle 医疗费用数据回归分析报告

## 1. 数据集说明

本次选择的 Kaggle 数据集是 **Medical Cost Personal Dataset**。

原始链接：`https://www.kaggle.com/datasets/d3lhomi10/medical-cost-personal-dataset`

该数据集中，每一行样本代表一位投保人的基本信息和对应医疗保险费用。目标变量是 `charges`，表示个人医疗保险费用，是一个连续型变量，因此适合做回归分析。

## 2. 变量含义

- `age`：年龄；
- `sex`：性别；
- `bmi`：身体质量指数；
- `children`：子女数量；
- `smoker`：是否吸烟；
- `region`：地区；
- `charges`：医疗保险费用，也是本次预测目标。

## 3. 为什么选择这个数据集

我选择这份数据，是因为它虽然规模不算特别大，但具有比较明确的现实业务含义。它同时包含数值变量和类别变量，例如年龄、BMI、子女数量是数值变量，性别、是否吸烟、地区是类别变量。因此它不是简单地直接套模型就结束，而是需要完成类别编码、异常值处理、标准化和模型解释。

## 4. 数据清洗与预处理

本次处理流程包括：

1. 删除重复样本；
2. 对 `bmi` 和 `charges` 使用 1% 和 99% 分位数进行缩尾处理；
3. 对类别变量进行 one-hot 编码；
4. 在每一折交叉验证中，只用训练集拟合缺失值填补器和标准化器，再作用于验证集，避免数据泄露；
5. 使用自定义 OLS 完成主模型训练；
6. 使用 RMSE、MAE、MAPE 评价模型表现。

## 7. VIF 共线性诊断

如果 VIF 明显大于 10，说明变量之间可能存在较强共线性。在真实数据中，共线性会影响单个变量系数的解释稳定性，即使模型预测效果还可以，也不能随便把每个系数都解释成严格的因果关系。

## 8. 真实数据推测结论

在这份医疗费用数据中，吸烟状态、年龄和 BMI 是比较容易解释的变量。其中吸烟状态的影响最明显，也最符合业务直觉。不过，真实数据和模拟数据不同，我们并不知道真正的数据生成机制，所以模型结果更多体现的是相关关系，而不能直接解释为因果关系。如果模型将来用于实际保险定价，我最担心的是样本代表性不足、异常值影响、以及不同群体之间可能存在的公平性问题。
