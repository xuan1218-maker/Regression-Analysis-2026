# NOAA 海洋浮标大数据正则化实验报告（Task B）

## 一、数据说明（海洋水文+气象，大样本）
- 数据集：NOAA 全球海洋浮标观测（1980–2025，Kaggle 下载）
- 样本量：3,317,076 行（逐小时观测）
- 特征数：8 个（海水温度、盐度、浪高、气温、气压、湿度、风速、风向等）
- 任务：预测 `water_temp_c`（海水温度，海洋核心环境变量）
- 特点：**强多重共线性**（如风速/浪高、气温/水温高度相关），完美适配正则化与变量筛选

## 二、模型设置与对比
### 1. OLS（无正则）
系数波动极大、不稳定，受共线性影响严重，泛化能力差，作为基准对照。
从结果可见，OLS 几乎无法识别出对水温有显著影响的变量，多数系数为0，完全无法反映真实海洋物理规律。

### 2. Ridge（L2 正则）
所有特征系数**平稳收缩**，有效缓解多重共线性，模型稳定性显著提升，适合高维共线海洋数据。
但 Ridge 无法自动剔除冗余特征，对无物理意义的变量仍保留非零系数，模型解释性有限。

### 3. Lasso（L1 正则）
自动将**冗余/无关特征系数压缩为 0**，实现嵌入式变量筛选，模型更简洁、可解释性强。
在本数据中，Lasso 成功筛选出 `air_temp_c`、`long_term_monthly_avg_temp`、`temperature_anomaly` 等关键变量，与海洋热交换的物理机制高度吻合。

### 4. ElasticNet（L1+L2 混合）
兼顾 Lasso 的稀疏性与 Ridge 的稳定性，**海洋高维共线数据最优选择**，避免 Lasso 在强共线时随机选变量的问题。
ElasticNet 不仅识别出了关键变量，还保留了部分次要相关特征（如 `pressure_hpa`、`longitude`），更完整地反映了海洋环境的复杂影响机制。

## 四、核心结论（海洋数据视角）
1. **正则化是海洋大数据建模必需**：OLS 在强共线水文/气象数据中完全不可用；
2. **Lasso 自动筛选关键海洋因子**：长期平均温度、温度异常、气温是影响水温的核心变量，符合海洋学常识；
3. **ElasticNet 综合最优**：在保留重要特征的同时抑制共线性，适合海洋环境预测任务；
4. **海洋变量强相关**：风速-浪高、气温-水温等天然共线，正则化是解决此类问题的标准方案。

## 五、业务关键影响因子（Top 5，按系数绝对值）
1. `long_term_monthly_avg_temp`（长期平均温度）——水温的主要基线水平
2. `air_temp_c`（气温）——海气界面热交换的直接驱动因素
3. `temperature_anomaly`（温度异常）——反映短期气候波动对水温的影响
4. `latitude`（纬度）——反映太阳辐射差异带来的温度梯度
5. `pressure_hpa`（气压）——通过影响海洋环流间接影响水温

以上因子与现实海洋学认知高度一致，模型具备良好的物理可解释性。
