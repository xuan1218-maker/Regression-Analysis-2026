# Week12 Summary

1. 三条核心结论
模型复杂度与误差：随着多项式次数升高，训练误差会一直下降，但测试误差先下降、后上升，这就是过拟合。
高复杂度带来高方差：模型越复杂，在不同数据集上的预测结果越不稳定，波动越大，说明泛化能力变差。
RMSE 对异常值更敏感：RMSE 会把大误差平方放大，而 MAE 只取绝对值，对异常点更稳健。
2. 最能代表过拟合的图
error_curves.png 最能体现过拟合。
在多项式次数达到 7 次之后，测试 RMSE 不再继续下降，反而开始波动上升，而训练误差还在继续下降。
这说明模型已经开始记住训练数据里的噪声，而不是学习真实规律，是非常直观的过拟合现象。
3. 什么时候用 RMSE / MAE
优先用 RMSE：希望严厉惩罚大错误的场景，比如金融预测、安全相关、医疗预测，一次大偏差代价很高。
优先用 MAE：数据里天然有很多异常值，或者希望指标更稳定、不容易被极端值带偏。
4. 为什么下一步要学正则化
因为模型太复杂 → 方差太高 → 容易过拟合。
正则化（Ridge / Lasso） 就是给模型加一个 “约束”，限制参数不要太大，强制降低模型复杂度，从而减少方差、缓解过拟合。
