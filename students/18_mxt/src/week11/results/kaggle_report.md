# kaggle_report.md 真实房价回归分析报告

## 一、数据集选择说明（B1）
- **数据集名称**：House Prices - Advanced Regression Techniques
- **原始链接**：https://www.kaggle.com/c/house-prices-advanced-regression-techniques
- **下载日期**：2026年5月24日
- **使用文件**：train.csv
- **预测目标**：SalePrice（房屋销售价格，连续变量，单位：美元）
- **样本业务含义**：每一行样本代表美国爱荷华州埃姆斯市2006-2010年间的一套住宅交易记录
- **选择理由**：该数据集包含真实世界的噪声、缺失值、异常值和复杂变量关系，非教学化简化数据，能够充分验证回归分析流程在真实场景下的有效性

## 二、数据预处理与特征工程（B2）
### 字段筛选与处理
- 原始数据集包含81个特征，基于业务知识和相关性分析，筛选出5个核心预测特征：
  1. `OverallQual`：房屋整体质量评分（1-10分）
  2. `GrLivArea`：地上居住面积（平方英尺）
  3. `GarageArea`：车库面积（平方英尺）
  4. `TotalBsmtSF`：地下室总面积（平方英尺）
  5. `YearBuilt`：房屋建造年份
- 删除含缺失值的样本（共删除11个样本，占比0.7%）
- 对所有数值特征进行2%双侧缩尾处理，去除极端豪宅和危房的影响
- 本次实验选用的特征均为数值型，无需类别变量编码

## 三、建模流程与工程实现（B2）
1. **全流程复用自定义utils组件**：自定义缺失值填充、异常值处理、标准化、OLS建模、指标计算、共线性诊断，仅使用sklearn KFold辅助交叉验证
2. **验证方式**：5折无泄露交叉验证，所有预处理内嵌于每一轮折内，杜绝数据泄露
3. **共线性诊断**：通过自定义VIF工具对真实数据进行共线性检测，结果显示所有特征VIF值均小于3，无严重共线性问题

## 四、实验结果与业务分析
### 模型性能指标
| 指标 | 真实运行结果 |
|------|--------------|
| RMSE | 37544.3077   |
| MAE  | 24781.4967   |
| MAPE | 14.7635%     |

### 结果分析
1. **整体效果**：14.76%的MAPE在真实房产预测场景中表现良好，模型泛化能力稳定
2. **稳定特征**：房屋整体质量、居住面积、车库面积与房价关联稳定，是房价核心驱动因素，符合房产交易业务常识
3. **不稳定特征**：房屋建造年份受市场行情、房屋翻新、老化程度干扰，系数波动较大，模型无法稳定捕捉其独立影响

## 五、问题诊断与上线风险（B4）
1. **数据问题**：真实数据无严重共线性，误差主要来源于未纳入模型的隐性特征（学区、装修、周边配套）和极端豪宅/危房样本
2. **业务误差解读**：平均2.48万美元的绝对误差，对应爱荷华州平均房价(约18万美元)的13.8%，是真实房产市场差异化大导致的正常现象，模型无系统性偏差
3. **上线风险**：对极端价格房屋预测精度差，泛化边界有限，上线需限制适用样本区间(房价5-50万美元)，增加极端值人工校验机制
