# 真实数据集实验报告（Task B）

## 一、数据来源与适用性
1. 数据集：kaggle_student.csv（学生考试成绩影响因素数据集）
2. 任务：回归预测 Exam_Score 考试成绩
3. 特征规模：总特征数 ≥15，包含学习、家庭、资源等多维度变量
4. 选择理由：特征间存在潜在共线性（如学习时长与家庭资源、参与度与成绩），是典型的高维、适合正则化与变量筛选的真实场景。

## 二、实验流程
1. 预处理：
   - 数值特征：使用自定义 CustomStandardScaler 标准化
   - 类别特征：独热编码（OneHotEncoder）
2. 划分：80% 训练集，20% 测试集
3. 模型：OLS、Ridge、Lasso、ElasticNet
4. 调参：5 折 GridSearchCV 超参数搜索
5. 评估：RMSE、MAE

## 三、测试集真实性能
| 模型 | RMSE | MAE |
|------|------|-----|
| OLS | 1.8034 | 0.4499 |
| Ridge | 1.8030 | 0.4492 |
| Lasso | 1.8030 | 0.4488 |
| ElasticNet | 1.8028 | 0.4488 |

结论：
- 正则化模型性能仅小幅优于 OLS，提升有限。
- 原因：数据集共线性程度中等，未导致 OLS 严重过拟合，正则化主要提升稳定性与可解释性。

## 四、Lasso 变量筛选与业务合理性
Lasso 通过 L1 正则自动剔除弱相关/冗余特征，保留核心变量：
- 保留关键因素：出勤率、学习时长、家庭资源、家长参与度等
- 剔除噪声/冗余特征：业务无关或重复信息的变量
筛选结果完全符合教育场景业务逻辑，剔除行为合理。

## 六、关键因素推荐依据
推荐以 **Lasso 结果**作为 Top5 关键影响因素：
1. Lasso 兼顾预测精度与稀疏性，结果简洁稳定
2. 自动消除共线性干扰，避免人为选择偏差
3. 输出可直接用于业务汇报，可解释性强
